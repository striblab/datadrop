Star Tribune Data Drop - Minneapolis' Digital Divide
================

by Frey Hargarten

Built using Mike Bostock's [D3](https://github.com/mbostock/d3), [NVD3](http://nvd3.org/), [jQuery](https://github.com/jquery/jquery) and [Mapbox.js](https://www.mapbox.com/mapbox.js/api/v2.2.2/). Powered by Google Sheets.

The files can be run in any live server setup. Just open the index.php or index.html file in your browser.
